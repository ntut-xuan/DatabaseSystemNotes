### File Structure Tree

`/pdf` 資料夾用來放置輸出的 `pdf` 結果，使用 `markdown` 進行編寫並呈現成 `pdf` 文件。

`/md` 資料夾用來放置 `markdown` 檔案，請在這邊進行編輯。

```
/pdf
| ------ DBSNotes_ch1.pdf
| ------ DBSNotes_ch2.pdf
| ------ DBSNotes_ch3.pdf
/md
| ------ DBSNotes_ch1.md
| ------ DBSNotes_ch2.md
| ------ DBSNotes_ch3.md
```
